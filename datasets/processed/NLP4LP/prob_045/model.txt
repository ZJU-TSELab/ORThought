Set:
1. ProductType
The set of product types, \( T = \{0, 1\} \) # 0 for Wraps, 1 for Platters
2. IngredientType
The set of ingredient types, \( I = \{0, 1\} \) # 0 for Meat, 1 for Rice

Parameter:
1. ProductionTime
[10, 8] # Minutes to make each product type
2. MinIngredientRequirement
[3000, 2500] # Minimum required units for each ingredient type
3. IngredientConsumption
[[5, 7], [3, 5]] # Units of each ingredient required per product of each type
4. MinWrapToPlatterRatio
3 # Minimum ratio of wraps to platters

Decision variable:
1. ProductCount
Integer variable, \( x_{t} \) for all \( t \in T \), representing the number of products of type \( t \) to make.

Objective:
Minimize the total production time.
min: \sum_{t \in T} ProductionTime[t] * x_{t}

Constraint:
1. Ingredient requirement constraint
\sum_{t \in T} IngredientConsumption[i][t] * x_{t} \geq MinIngredientRequirement[i], for all \( i \in I \)
2. Wrap to platter ratio constraint
x_{0} \geq MinWrapToPlatterRatio * x_{1}
3. Non-negativity
x_{t} \geq 0, for all \( t \in T \)

Type:
Integer, linear, linear
