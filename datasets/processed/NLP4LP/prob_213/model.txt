Set:
1. TestType
The set of test types, \( I = \{0, 1\} \) # 0 for Temperature Check, 1 for Blood Test

Parameter:
1. TimePerTest
[2, 10] # Time in minutes for each test type
2. MinBloodTests
45 # Minimum number of blood tests to be conducted
3. TempCheckRatio
5 # The number of temperature checks must be at least 5 times the number of blood tests
4. TotalStaffMinutes
22000 # Total available staff minutes

Decision variable:
1. NumTests
Integer variable, \( x_{i} \) for all \( i \in I \), representing the number of each test type to be performed.

Objective:
Maximize the total number of tests performed.
max: \( \sum_{i \in I} x_{i} \)

Constraint:
1. Minimum blood tests constraint
\( x_{1} \geq MinBloodTests \)
2. Temperature check ratio constraint
\( x_{0} \geq TempCheckRatio \cdot x_{1} \)
3. Staff minutes constraint
\( \sum_{i \in I} TimePerTest[i] \cdot x_{i} \leq TotalStaffMinutes \)
4. Non-negativity and Integer
\( x_{i} \geq 0 \) and integer, for all \( i \in I \)

Type:
Integer, linear, linear
ILP
