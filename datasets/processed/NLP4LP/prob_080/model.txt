Set:
1. MachineType
The set of machine types, \( I = \{0, 1\} \) # 0 for Machine A, 1 for Machine B

Parameter:
1. ProductionRate
[30, 50] # Items made per day by each machine type
2. EnergyConsumption
[100, 120] # kWh consumed per day by each machine type
3. MinProduction
1000 # Minimum total items to be produced per day
4. MaxEnergy
3000 # Maximum total kWh of electricity available per day
5. MaxBPercentage
0.3 # Maximum percentage of machines of type B
6. MinAMachines
5 # Minimum number of machines of type A

Decision variable:
1. MachineCount
Integer variable, \( x_{i} \) for all \( i \in I \), representing the number of machines of type \( i \) to be used.

Objective:
Minimize the total number of machines.
min: \( \sum_{i \in I} x_{i} \)

Constraint:
1. Production constraint
\( \sum_{i \in I} ProductionRate[i] \cdot x_{i} \geq MinProduction \)
2. Energy constraint
\( \sum_{i \in I} EnergyConsumption[i] \cdot x_{i} \leq MaxEnergy \)
3. Machine B proportion constraint
\( x_{1} \leq MaxBPercentage \cdot (x_{0} + x_{1}) \) which simplifies to \( 7 \cdot x_{1} \leq 3 \cdot x_{0} \)
4. Minimum machine A constraint
\( x_{0} \geq MinAMachines \)
5. Non-negativity
\( x_{i} \geq 0 \), for all \( i \in I \)

Type:
Integer, linear, linear
