Set:
1. DrinkType
The set of drink types, \( T = \{0, 1\} \) # 0 for Drink A, 1 for Drink B
2. VitaminType
The set of vitamin types, \( V = \{0, 1, 2\} \) # 0 for Vitamin A, 1 for Vitamin D, 2 for Vitamin E

Parameter:
1. VitaminKContent
[4, 12] # Units of Vitamin K per cup of each drink type
2. VitaminRequirements
[150, 300, -1] # Minimum units of Vitamin A and D, -1 for Vitamin E as it's an upper bound
3. VitaminE_UpperBound
400 # Maximum units of Vitamin E
4. VitaminContent
[[8, 15], [6, 2], [10, 20]] # Units of each vitamin per cup of each drink type

Decision variable:
1. DrinkCups
Continuous variable, \( x_{t} \) for all \( t \in T \), representing the number of cups of drink type \( t \) to use.

Objective:
Minimize the total amount of Vitamin K.
min: \sum_{t \in T} VitaminKContent[t] * x_{t}

Constraint:
1. Vitamin A requirement
\sum_{t \in T} VitaminContent[0][t] * x_{t} \geq VitaminRequirements[0]
2. Vitamin D requirement
\sum_{t \in T} VitaminContent[1][t] * x_{t} \geq VitaminRequirements[1]
3. Vitamin E constraint (upper bound)
\sum_{t \in T} VitaminContent[2][t] * x_{t} \leq VitaminE_UpperBound
4. Non-negativity
x_{t} \geq 0, for all \( t \in T \)

Type:
continuous, linear, linear
